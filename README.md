# SQL Sales Data Analysis

This project uses SQL to analyze a retail sales dataset and extract business insights.

## Tools Used
- SQL (SQLite)
- Data Aggregation and Filtering
- Grouping and Sorting Queries

## Dataset
The dataset contains retail sales information including product details, order dates, regions, and sales values.

## Analysis Performed

The following queries were used to analyze the dataset:

- Total sales calculation
- Sales by product category
- Sales by region
- Top 10 products by revenue
- Average order value

## Key Insights

- Technology category generates the highest total revenue.
- West and East regions show the strongest sales performance.
- A small group of products contributes significantly to overall revenue.
